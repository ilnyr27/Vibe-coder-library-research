# 7. Доступность / Accessibility

← [[VAIBCODER-Index]] | → [[08-Forms]]

- Семантический HTML (`<nav> <main> <button>`), не `div` на всё.
- Навигация с клавиатуры + видимый фокус.
- Контраст по WCAG AA (≥ 4.5:1 для текста).
- `alt` у изображений; ARIA только где реально нужно.
- Примитивы shadcn/Radix доступны по умолчанию — не ломай их.
