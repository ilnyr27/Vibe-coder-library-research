# 🇷🇺 Деплой — Timeweb

← [[VAIBCODER-Index]] | → [[Deploy-Vercel]] · [[Deploy-VPS]]

**Когда:** запасной путь, если Vercel недоступен из РФ; серверы в России (152-ФЗ).

**Timeweb Cloud App Platform:**
1. Подключи GitHub/GitLab/Bitbucket.
2. Тип приложения — Next.js + версия Node.
3. Автодеплой по последнему коммиту.
4. Бесплатный автопродлеваемый SSL.
5. Привязка/покупка домена — раздел «Домены и SSL».

**Нюансы:** оплата картами РФ/СБП; базовый тариф ~1 ГБ NVMe; **SSR нельзя отключить после деплоя** — выбирай тип верно сразу.
